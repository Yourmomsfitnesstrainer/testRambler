1. На машине должен быть установлен nodeJS
2. Выполнить в директории "npm i"
3. Если хотите, можно отключить открытие Chromium, тогда тесты выполнятся в headless режиме. Для этого
в документе "codecept.conf.js" закомментируйте 7ую строчку - "show: true,".
4. Запустить тесты командой "npm run test"
5. При первом запуске в директрии проекта будет создана папка "LogScreen". В ней будут отчёты о выполнении тестов
в json и html форматах, а также скрин из сценария проверки новых писем. 
6. Открыть html формат через любой удобный браузер.
